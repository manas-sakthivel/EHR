Verification Process
  1. The system calculates the SHA-256 hash of your uploaded file
  2. It retrieves the original hash from the blockchain
  3. Both hashes are compared to determine if the file has been tampered with
  4. The verification result is logged on the blockchain for audit purposes
How Verification Works
Hash Calculation

The system calculates a SHA-256 hash of your uploaded file.

Blockchain Lookup

It retrieves the original file hash stored on the blockchain.

Hash Comparison

Both hashes are compared to detect any modifications.

Result Logging

The verification result is permanently recorded on the blockchain.

Verification Results
Integrity Verified

File hash matches the blockchain record. File has not been tampered with.

Integrity Failed

File hash does not match. File may have been modified or corrupted.

Verification Logged

All verification attempts are recorded on the blockchain for audit purposes.
